# Como gerar o .exe Windows sem ter Windows

Este projeto usa GitHub Actions para gerar o executável Windows.

## Passo a passo

1. Suba o projeto para o GitHub.
2. Vá na aba Actions.
3. Abra "Build Windows Release".
4. Clique em "Run workflow" ou faça um push na branch main/master.
5. Espere terminar.
6. Baixe o artifact chamado "AcougueDoLeleco-Windows".
7. Extraia o zip.
8. Entregue a pasta extraída para a dona.

## Importante

A dona NÃO precisa instalar Flutter.

Ela precisa receber a pasta do programa pronta e abrir o arquivo .exe.

Não entregue só o .exe separado. Entregue a pasta inteira da build.
